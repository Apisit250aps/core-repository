# v0.2.2

Release date: 2026-04-25

## Summary

- Patch release เพื่อแก้การใช้งาน controller method แบบส่งตรงเข้า Hono route handler
- รองรับรูปแบบ `app.get('/api/posts', postController.get)` โดยไม่ทำให้ `this` หลุดจาก controller instance
- ลดความจำเป็นในการเขียน wrapper function เพิ่มแค่เพื่อรักษา context ของ method

## Bug Fixes

- Bind CRUD handlers (`get`, `getById`, `create`, `update`, `delete`) ใน `BaseController` constructor
- แก้ runtime error `Cannot read properties of undefined (reading 'error')` ที่เกิดเมื่อส่ง method reference ของ controller เข้า `app.get(...)` ตรงๆ
- ทำให้ helper methods อย่าง `this.ok(...)`, `this.error(...)`, และการเข้าถึง `this.repository` ใช้งานได้ตามปกติภายใน handler

## Compatibility Notes

- โค้ดที่เคยใช้ wrapper เช่น `app.get('/api/posts', (c) => postController.get(c))` ยังใช้งานได้ตามเดิม
- release นี้เพิ่มความเข้ากันได้กับการใช้งาน Hono โดยตรง แต่ไม่ได้เปลี่ยน public API ของ controller base class

## Breaking Changes

None. This release is intended to be backwards-compatible with v0.2.1.
