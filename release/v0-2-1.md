# v0.2.1

Release date: 2026-04-25

## Summary

- Patch release เพื่อ align dependency ของ `uuid` ไปที่ v14 ทั้ง root package และ core package
- ปรับ lockfile ให้ตรงกับ dependency set ล่าสุด
- ไม่มีการเปลี่ยน public API, import path, หรือ export surface จาก v0.2.0

## Dependency Updates

- Root package (`@aps/next-api`) เปลี่ยน `uuid` จาก `^13.0.0` เป็น `^14.0.0`
- Core package (`@aps/next-api-core`) เปลี่ยน peer dependency ของ `uuid` จาก `^13.0.0` เป็น `^14.0.0`
- Refresh `package-lock.json` ให้สะท้อนเวอร์ชัน dependency ปัจจุบัน

## Compatibility Notes

- โค้ดที่ใช้งานผ่าน `@aps/next-api` และ `@aps/next-api-core` ยังใช้ import เดิมได้
- Release นี้เหมาะสำหรับโปรเจกต์ที่ต้องการใช้งาน `uuid@14` ให้ตรงกับ ecosystem ปัจจุบัน

## Breaking Changes

None. This release is intended to be backwards-compatible with v0.2.0.
